Nous avons décidé de faire ce site web sur le système solaire, en présentant chaque planète de notre système ainsi que Cérès et Pluton qui sont des planètes naines. Pour faciliter notre travail, nous avons créé un git. Nous nous sommes repartis les taches équitablement lors de ce projet

Fonctionalité :

  • L'utilisateur peut cliquer sur les planètes sur la page d'accueil pour être redirigé vers une page planète.br.
  • L'utilisateur peut cliquer sur les planètes sur les pages planète pour faire apparaitre le planisphère de la planète.
  • L'utilisateur peut chercher des informations géographiques sur la planète en passant sa souris sur le planisphère.
  • L'utilisateur peut cliquer sur l'information 'Plus de photos' pour ouvrir un carrousel.
  • L'utilisateur peut revenir sur la page d'accueil en cliquant sur le système solaire sur le titre de la page.
  • L'utilisateur peut passer de planète en planètes en cliquant sur les fusé sur le header.
